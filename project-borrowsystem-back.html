LABORATORY EQUIPMENT BORROWING SYSTEM (BACKEND)

ABOUT THE PROJECT

github repository

Web application borrowing system backend API for laboratory equipments, tools, and components such as Digital Trainer, Microcontrollers, Pliers etc.

PROJECT DETAILS

This backend system uses Express JS framework and is run via NodeJS
This backend system uses JSONWEBTOKEN (JWT) as middleware in verifying the user
This backend system uses mongoose for schema modelling, validation and creating queries for MongoDB Database
