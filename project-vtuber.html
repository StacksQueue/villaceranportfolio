VTUBER BY KALIDOKIT

ABOUT THE PROJECT

github repository

Kalidokit is a face, pose, and hand tracking calculator for VRM and 2D models develop by yeemachine.

In the tutorial, this only uses CDN links so I made possible in angular. An experience in THREEJS is a great advantage in developing this web app.

PROJECT DETAILS

The face, eyes, hands, and body body movements are coordinated with the help of Mediapipe/Tensorflow.js
The mouth is coordinated from the camera and the loaded VRM model as Mediapipe/Tensorflow tracks face compatible with Facemesh
The hand is coordinated from the camera and the loaded VRM model as Mediapipe/Tensorflow tracks hands compatible with Handpose
The eyes are coordinated from the camera and the loaded VRM model as Mediapipe/Tensorflow tracks face compatible with Facemesh
